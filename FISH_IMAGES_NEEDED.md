# Fish Images Needed

Save each image as the filename listed below into `public/assets/fish/`.
Ideal specs: transparent PNG, roughly square, side-profile facing right.
Any size works — the game scales them to 48-64px with pixelated rendering.

Good free sources:
- **Wikimedia Commons** (public domain / CC): https://commons.wikimedia.org
- **FCIT ClipArt ETC** (free educational use): https://etc.usf.edu/clipart/
- **iNaturalist** (CC photos): https://www.inaturalist.org
- **NOAA Fish Photos** (US govt, public domain): https://www.fisheries.noaa.gov

## Tips
- Wikimedia scientific illustrations are great — they're clear, accurate, and usually public domain
- If you find a photo with a white background, that works too
- Don't worry about exact size — anything from 100px to 1000px is fine, the code handles scaling
